# Dataset

Este proyecto usa el dataset publico **Telco Customer Churn**, basado en datos de muestra de IBM.

Opciones de descarga:

- Fuente recomendada: [Kaggle - Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)
- Respaldo reproducible usado por el script: `https://raw.githubusercontent.com/treselle-systems/customer_churn_analysis/master/WA_Fn-UseC_-Telco-Customer-Churn.csv`

Para descargarlo automaticamente:

```bash
python src/data_loader.py
```

El archivo quedara en `data/telco_churn.csv`.
